1688 比赛中的配对次数

按题意模拟，每次除以2。n向下取整，结果向上取整并求和。

1689 十-二进制的最少数目

构造题。结果等价于找到结果中的最大数字。

注：存在进位的必要条件是至少有10个数。我们发现如果有10个数，必然可以有更小的结果。所以结果必然出现在1-9之间，此时不存在进位，需要至少n个数目，n为结果数中的最大数字。

1690 石子游戏

给定一排石子，每个石子拥有正整数分。A和B轮流从最左或者最右取一颗石子，并且得到其余石子分数和的分数。石子取光时游戏结束。A和B为了取得最高的分数均采取了最优策略。请返回游戏结束时两人的分数差。

题解

首先我们发现，在任何时刻，游戏的残存局面必然是原串的子串，因为只能从左右两侧取子。

不失一般性，我们假设dp(i,j)为当前游戏进行到局面为i到j的子串时，当前先手与后手的最优分数差。不难发现，只有两种选择：取最左侧或者最右侧的石子。容易得到状态转移：

\[ dp(i,j) = max(-dp(i+1, j) + sum(i+1, j), -dp(i, j - 1) + sum(i, j - 1)) \]

\(dp(0, n-1)\)为答案，边界条件为\(dp(i,i)=0\)。

状态数为\(O(n^2)\)，求和可以用前缀和\(O(1)\)取得。我使用了记忆化搜索的写法，获得了总\(O(n^2)\)的时间复杂度。也可以使用自底向上的写法，以子串长度递增顺序遍历所有子串求值。

1691 堆叠长方体的最大高度

给定n个长方体，两个长方体当且仅当其中一个的长宽高均不大于另一个时，才能将小的堆叠到大的上面。

与以往此类问题不同的是，该题中所有长方体都允许任意旋转。

问最大堆叠高度是多少？

题解

该题如果不包含旋转，可以简单使用对一个维度排序加上动态规划的方式求解。

该题的难点是处理旋转。如果因为旋转的引入导致了原有偏序不存在，那么将无法使用动态规划求解。

好在我们可以首先将长方体翻转到长 < 宽 < 高的状态，再对一个维度进行从小到大排序。此时我们发现，处于后面的长方体必然无法被放在前面的长方体上方，无论怎么旋转。那么动态规划的状态就呼之欲出：\(dp(i,j)\)表示以第i个长方体作为底，并且长方体i的旋转状态是j的时候，最高能叠多少。

状态转移方程为

\[ dp(i,k) = max_{j < k} (dp(j,u), \text{ if }box_{j,u} \leq box_{i,k}) \]

时间复杂度\(O(n^2)\)

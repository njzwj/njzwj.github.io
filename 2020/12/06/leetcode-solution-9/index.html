1681 - Minimum Incompatibility

给定任意序列和正整数\(k\)。将其分为\(k\)个等大的子集，并且每个子集不包含相同元素。定义每个子集的最大最小值差为不相容性。现在要求最小的不相容性的和。不存在的情况输出1。数据范围：数组长度不超过\(16\)。

题解

假设序列长度为\(n\)。枚举预处理出所有的\(n/k\)长合法子集，并且统计他们的不相容性。然后使用状态压缩动态规划，将子集的选取状态作为状态，转移方程为：

\[ dp[status] = min{dp[status\setminus set] + incompatibility[set]} \]

1675 - Minimize Deviation in Array

给定由\(n\)个正整数组成的数组nums。现在对元素执行任意次数的两类操作：

  1. 如果元素是偶数，除以2

  2. 如果元素是奇数，乘2

定义数组的偏移量是数组两个元素之间的最大差值。问进行若干次这样操作后最小可能的偏移量。

题解

我们观察到，如果一个数\(a=2^k * p\)，那么它可能的取值为\(a'=2^x * p (0\leq x \leq k)\)。如果一个数\(a=p\)，那么它只可能取\(a\)或\(2a\)。

此时我们考虑这样一套策略：从每个数最大的状态开始挑选一些数将其缩小。如果某个情况下最大数为奇数，那么此时的数组拥有最小的偏移量。

我们使用堆来处理这一策略。每次取得堆顶的最大元素将其除以2，放回去。

1674 - Minimum Moves to Make Array Complementary

给定长度为偶数\(n\)的数组和limit，对数组中每个元素，可以将它改为1到limit间的任意整数。一个数组如果所有的a[i]和a[n-i-1]的和均相等，那么我们称数组互补。问最少要多少次操作能将数组变为互补的。

题解

观察任意数对a和b，不失一般性，假设a > b。我们从2开遍历目标的a+b和x，得到最小操作次数\(f(a,b)\)。

\[ f(a,b)=\left\{ \begin{aligned} 2&, x \leq b \\ 1&, b + 1 \leq x \leq a + b - 1 \\ 0&, x = a + b\\ 1&, a + b + 1\leq x\leq a + limit\\ 2&, a + limit + 1 \leq x \end{aligned} \right. \]

使用差分数组维护遍历期间的总操作代价，从小到大遍历目标和的取值即可找到最小次数。

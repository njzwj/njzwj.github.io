1787 使所有区间的异或结果为零

我们发现任意k长区间异或和相等需要序列满足k循环性质，也就是说结果串中a[i] = a[i + k]。证明也很简单，a[i]^a[i+1]^...^a[i+k-1] = a[i+1]^a[i+2]^...^a[i+k]，另异或满足交换律，消去两边同项得a[i]=a[i+k]。由于模k下标相同的元素一定会变成相同元素，所以我们只用考虑k个下标即可。

在每一个位置我们可以将所有元素变为某个已有的元素，或者是变成一个完全没有出现过的新元素。现在规定状态dp[i, s]表示前i项异或和为s的时候最小的操作次数。

那么对于所有可能的s，我们都可以由i-1位置的最优状态s'，将i位置变为s'^s得到，新增代价为位置i的元素总数。

我们还可以将i处变为某个已经出现过的数，此时我们需要遍历所有i-1的状态s，并且相应更新现在的状态。

时间复杂度\(O(2^D K + 2^D N) = O(2^D (K + N))\)。

1792 最大平均通过率

想了很多全局优化的方法，后来发现这就是个贪心。如果仔细观察一下将分数a/b同时加上一个常数c，随着c的增加，边际效应递减。具体地说：

\[ \frac{a + c + 1}{b + c + 1} - \frac{a + c}{b + c} \geq \frac{a + c + 2}{b + c + 2} - \frac{a + c + 1}{b + c + 1} \]

由于该题要求优化所有分数的均值（与优化和等价），那么每次找到增加一个常数使得分数增加绝对值最大的操作一定是最优解。使用堆维护当前最值得进行的操作即可。时间复杂度\(O(K\log{N})\)。

1793 好子数组的最大分数

数组的得分由长度和最小值的乘积决定。我们先回忆一下区间最小值的特点——固定一个端点，它的大小随另一个端点具有单调性。区间长度与之有相反的单调性。

由于子区间已经到包含k，可以从k开始向左右扩展双指针，尽量保持两边区间最小值均衡。期间更新答案。
